# 2026 世界杯淘汰赛预测网站

## 项目概述
基于 Dixon-Coles 泊松模型 + Elo 评级 + 赛事 xG + 球员数据的 2026 世界杯淘汰赛预测系统。
从 16 强到冠军的完整晋级概率，蒙特卡洛 10,000 次模拟。

## 文件结构
```
├── index.html          # 单文件版本（浏览器直接打开，无需服务器）
├── app.py              # Flask 主应用（6 页面路由 + 2 API）
├── model.py            # Dixon-Coles 模型 + 蒙特卡洛引擎
├── data.py             # 全部数据（16队 + 25球星 + 12组 + 32强赛果）
├── data.json           # 实时赛果数据（定时任务自动更新）
├── requirements.txt    # flask + gunicorn
├── static/style.css    # 样式
└── templates/          # Jinja2 模板
    ├── base.html       # 布局模板（含 60s 自动轮询）
    ├── index.html      # 淘汰赛主页（R16 预测 + 对阵树）
    ├── groups.html     # 小组赛积分榜
    ├── teams.html      # 球队排名 & 夺冠概率
    ├── team.html       # 球队详情页
    └── match.html      # 单场比赛详情页
```

## 预测模型
- **Dixon-Coles 双变量泊松** — 两队进球数建模，含低比分相关性修正 ρ=-0.06
- **四维加权综合评分：赛事表现 55% + Elo 15% + 球员质量 20% + 动量 10%**
- 蒙特卡洛：每次模拟含 90 分钟 + 加时赛 + 点球决胜
- 点球胜率基于球队综合评分差异
